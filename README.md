# UST_Lecture_AgenticAI


## Phase 1: Onboarding & Literacy

**Week: 1 (3.2) Orientation: Agentic AI as R&D Collaborators**

- - [Lecture] Course Overview: The shift from "Using Tools" to "Directing Agents"
  
- - [Practice] Environment Setup: Google AI Studio + Google Antigravity + Python + Github
  
- - [Discussion] Is AI a research assistant or a crutch? Defining the boundary



**Week 2 (3.9) Understanding of Large Language Models**

- [Lecture] The Brain of the Agent: Capabilities, Limitations, and Security (Cloud vs. On-premise)

- [Practice] API Connection: Calling LLMs via Python & Setting up Ollama for secure local use

- [Discussion] The "Stochastic Parrot" Problem: How much can we trust the probabilistic answers?



**Week 3 (3.16) Structured Directing based on Prompt Engineering**

- [Lecture] The Art of Instruction: System Prompts and Persona Design for Researchers

- [Practice] CLI Agent: Building a command-line interface agent with a specific "Research Persona",

- [Discussion] Managing Expectations: What AI can do vs. what it should never do



**Week 4 (3.23) AI Orchestration based on Tools**

- [Lecture] Giving Hands to AI: Understanding Function Calling and Tool Use

- [Practice] Custom Tools: Connecting Python functions (Calculator, Search) to your agent

- [Discussion] The Director's Role: If AI uses the tools, what is the human's unique contribution?


## Phase 2: Workflow Automation & Design

**Week 5 (3.30) Human-AI Agent Interaction Design**

- [Lecture] UX for Research: How to design interfaces for collaboration

- [Practice] Web UI: Building a deployable agent app using Streamlit/Gradio

- [Discussion] Core Competency: In an era where AI conducts experiments and writes papers, what is the "irreplaceable" skill of a researcher?



**Week 6 (4.6) Automation of Data Input and Meta Information Extraction**

- [Lecture] The Input Pipeline: Automating literature review and data collection

- [Practice] Auto-Scraper: Building a pipeline to collect papers and extract meta-data without reading

- [Discussion] The Risk of "Not Reading": How to maintain deep insight while automating data intake



**Week 7 (4.13) Automation of Data Output and Discovery**

- [Lecture] The Output Pipeline: Drafting reports and visualizing experimental results

- [Practice] Auto-Drafting: Converting raw CSV data into a "Results & Discussion" draft

- [Discussion] Authorship & Ethics: Who is responsible if the AI-generated hypothesis is flawed?



**Week 8 (4.20 = Mid-term Evaluation) Proposal of My Agent System**

- [Practice] Mid-term Presentation: Proposal of Agentic AI System (3 to 5-minute pre-recorded video)

- [Discussion] Peer-Review: Assessing the feasibility and value of colleagues' proposals


## Phase 3: Management & Reliability

**Week 9 (5.4) Design of Task and Workflow**

- [Lecture] Algorithmic Thinking: Designing robust workflows (LangGraph basics)

- [Practice] Flow Architecture: Designing a multi-step research logic (Plan -> Execute -> Review)

- [Discussion] Ambiguity vs. Clarity: Why do agents fail when human instructions are vague?



**Week 10 (5.11) Evaluation of Output from Agent**

- [Lecture] Quality Control: Establishing metrics for AI performance

- [Practice] Evaluation System: Designing a rubric and running statistical tests on agent outputs

- [Discussion] The Consensus Dilemma: Which is more persuasive—1,000 AI agents or 10 human experts?



**Week 11 (5.18) Self-feedback Loop Agent System**

- [Lecture] Metacognition: How to make agents critique and fix their own errors

- [Practice] Self-Correction Pipeline: Implementing a "Reflection" node in the workflow

- [Discussion] Nurturing AI: Is "teaching" AI to fix itself a new form of mentorship?



**Week 12 (5.25) Memory and Retrieval-augmented Generation (RAG)**

- [Lecture] Long-term Context: Vector databases and maintaining research history

- [Practice] RAG Pipeline: Building an agent that "remembers" your past papers and lab notes

- [Discussion] Selective Memory: Should AI remember failed experiments? The value of negative data



## Phase 4: Collaboration & Future

**Week 13 (6.1) Multi-agent System**

- [Lecture] Team Dynamics: Orchestrating multiple agents with different roles (e.g., Writer vs. Reviewer)

- [Practice] Role-Playing Simulation: Building a 2-agent system that debates a topic

- [Discussion] Leadership: How to manage a digital workforce of conflicting agents



**Week 14 (6.8) Human in the Loop (HITL) in Agentic AI System**

- [Lecture] Safety & Audit: Designing "Stop" buttons and approval processes

- [Practice] Audit Interface: Inserting a human approval step before the final output

- [Discussion] The Gatekeeper: How to identify and filter out villain or biased agents



**Week 15 (6.15) Standardization and Expansion Capability**

- [Lecture] Connectivity: Model Context Protocol (MCP) and connecting to external labs

- [Practice] Interoperability: Designing an agent that can talk to other students' agents

- [Discussion] Sustainability: How to build an AI-assisted research ecosystem that is energy-efficient and ethical



**Week 16 (6.22 = Final Showcase) Final Evaluation: Demo Day**

- [Activity] Live Demonstration: Showcasing the developed Agentic AI System (3 to 5-minute pre-recorded video)

- [Report] "The Director's Report": A reflection on the management process, failures, and ethical considerations

- [Discussion] Peer-Review: Assessing the developed system based on technical completeness and sustainability
